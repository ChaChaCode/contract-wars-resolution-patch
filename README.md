# Contract Wars — Tweaks Patch

[![Download](https://img.shields.io/badge/%E2%AC%87%20%D0%A1%D0%9A%D0%90%D0%A7%D0%90%D0%A2%D0%AC%20%2F%20DOWNLOAD-latest-c69c3c?style=for-the-badge)](https://github.com/ChaChaCode/contract-wars-resolution-patch/releases/latest)

Набор улучшений для клиента Contract Wars. Патчи выбираются галочками в окне программы:

- **Разрешение** — снимает ограничение 1920×1080, открывает 2K / 4K / 8K и все разрешения монитора.
- **Надёжный клик входа** — чинит баг, когда клик на экране загрузки «не срабатывает» и приходится перезаходить.
- **Автоспавн** — сразу закидывает в бой без клика «ЛКМ для вступления» (Deathmatch и Team Elimination).
- **FOV + выбор экрана** — добавляет прямо в настройки игры: ползунок **FOV** (угол обзора 60–110°) и чекбокс **«Полный экран»**. Прицеливание не затрагивается.
- **Borderless** — запуск в окне без рамки на весь монитор (`BORDERLESS.vbs`). Alt-Tab мгновенный, можно класть окна поверх игры, не сворачивается при переключении на другой монитор — удобно для нескольких мониторов.

*A tweaks patch for the Contract Wars client: unlock 2K/4K/8K resolution, fix the unreliable "click to start" bug, and optional auto-spawn (no click to join). Toggle each in the app window.*

## Как пользоваться

1. **[Скачать ZIP](https://github.com/ChaChaCode/contract-wars-resolution-patch/releases/latest)** → распаковать целиком (не запускать из архива).
2. Запустить **`PATCH.vbs`** — сразу откроется окно, без чёрных консолей.
3. Выбрать папку игры (находится сама) → отметить нужные патчи → **«ПРИМЕНИТЬ ПАТЧ»**.
4. Запустить игру.

В распакованной папке всего два файла для вас: **`PATCH.vbs`** (запуск) и **`README.md`** (эта инструкция). Всё остальное лежит в папке `engine` — её трогать не нужно.

**Откат:** кнопка «Откатить» в окне — возвращает оригинальный `Assembly-CSharp.dll` из бэкапа.

### Окно без рамки (borderless)

Запустите **`BORDERLESS.vbs`** — игра откроется в окне без рамки на весь монитор. Такое окно **не сворачивается** при клике на другой экран (в отличие от обычного полноэкранного), поэтому удобно работать на нескольких мониторах. Это только способ запуска — код игры не меняется.

**Если патч не находит место в коде** (другая сборка клиента): нажмите **«Диагностика»** — лог сразу копируется — и пришлите его в Telegram **[t.me/Moxy1337](https://t.me/Moxy1337)**.

## Что делает каждый патч

| Патч | Суть |
|------|------|
| Разрешение | Поднимает потолок в `Utility.FixResolution` и в фильтре списка настроек. |
| Клик входа | На экране «READY!» заменяет одноразовую проверку нажатия на удержание — клик перестаёт «теряться». |
| Автоспавн | В коде спавна (Deathmatch, Team Elimination) вход в бой больше не требует клика. Таймер респауна и выбор стороны соблюдаются. Tactical Conquest / Target Designation используют выбор точки на карте — не затрагиваются. |
| FOV + экран | Добавляет в настройки игры (Видео/Аудио): ползунок **«FOV (угол обзора)»** (60–110°) и чекбокс **«Полный экран»** (окно на весь монитор ↔ оконный). Значения хранятся локально (не на сервере). Прицеливание, модель оружия и оптика — не затрагиваются. Требует файл `CWScreen.dll` (кладётся в игру автоматически). |
| Borderless | Запуск через `BORDERLESS.vbs` в окне без рамки на весь монитор (флаг движка Unity `-popupwindow`). Alt-Tab мгновенный, окна кладутся поверх, не сворачивается на мультимониторе. Файлы игры не меняются — только способ запуска. |

Патчи DLL ищут места **по именам методов и байт-паттернам**, поэтому переживают разные сборки клиента; при несовпадении файл не меняется. Оригинал сохраняется в `Assembly-CSharp.dll.orig.bak`.
